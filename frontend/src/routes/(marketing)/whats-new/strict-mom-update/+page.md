## What's New?

We've been working hard to make Cubrain even more reliable. This update focuses on better error handling and a smoother subscription experience.

### ✨ Key Features

- **Grace Period Logic**: Users now have a 3-day grace period after their subscription expires before losing access to Pro features.
- **Empty File Validation**: We've added strict validation for file uploads. No more "ghost" decks from empty PDFs!

### 🐛 Bug Fixes

- Fixed a layout shift on the landing page when the launch banner was active.
- Improved the real-time counter for the early bird discount.

### 🛠️ Maintenance

- Updated Supabase Auth integration for better session management.
- Refactored the global layout to centralize SEO and analytics.

> [!TIP]
> You can now see your subscription status in the settings page with more detail!
